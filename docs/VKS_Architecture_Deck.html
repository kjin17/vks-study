기존 가상화 대비 — 무엇이 다른가

VKS는 기존 가상화 인프라 전체를 쿠버네티스화(K8s-Native)하여 관리한다
  • 기존 가상화 — 중앙에서 가상 머신(VM)이라는 컴퓨팅 리소스를 할당·관리하는 데 초점. VI Admin이 vCenter로 VM 단위를 운영한다.
  • VKS 방식 — 인프라 전체(컴퓨트·네트워크·스토리지)를 쿠버네티스 선언적 API(YAML)로 논리적 단위로 동적 할당·관리. VI Admin과 개발자(DevOps) 모두에게 익숙한 인터페이스를 제공한다.
    • 별도의 외부 도구나 사일로 없이 vSphere 인프라 위에서 직접 K8s 클러스터를 배포·스케일링
    • VM(전통 워크로드)과 컨테이너(클라우드 네이티브)를 하나의 플랫폼에서 프로비저닝·운영
  • 핵심 — "가상화 위에 쿠버네티스를 얹는 것"이 아니라, 가상화 인프라 자체를 쿠버네티스로 표현·제어한다.

Supervisor가 배포하는 3가지 워크로드

전통 VM부터 하이퍼바이저 네이티브 Pod, 완전한 게스트 클러스터까지
VM Services

전통 VM을 K8s API로

  • 기존 VM을 K8s API(YAML)로 관리
  • VM Operator CRD (VirtualMachine) 사용
  • VM 생명주기 자동화
vPod

vSphere Pods

  • ESXi 하이퍼바이저 위에서 직접 실행되는 네이티브 Pod
  • VM 격리 수준 보안 · 별도 Worker VM 불필요
  • 빠른 프로비저닝 · 높은 성능
VKS Cluster

Guest / TKG 클러스터

  • 독립적인 업스트림 K8s 클러스터 프로비저닝
  • CP + Worker 모두 VM · 표준 K8s API 완전 호환
  • 멀티 테넌트 · Helm/kubectl 사용

Supervisor Service & VKS Service

플랫폼 컨트롤 플레인 서비스 vs 게스트 클러스터 수명주기 서비스
Supervisor Service

플랫폼 컨트롤 플레인

  • vSphere 위에 배포되는 K8s 컨트롤 플레인 서비스
  • 개발자·DevOps에 K8s API 엔드포인트 제공 — VM·컨테이너·스토리지·네트워크를 요청
  • IT 관리자는 vCenter에서 보안·수명주기·리소스 정책을 중앙 집중 관리
VKS Service

게스트 클러스터 수명주기

  • 게스트 클러스터의 생성·스케일·업그레이드 전체 수명주기를 자동화
  • CNCF 인증 표준 K8s 클러스터를 온디맨드 제공
  • 퍼블릭 클라우드와 동일한 셀프 서비스 경험

vSphere Kubernetes Release (VKR / TKr)

Broadcom이 테스트·호환성 검증을 완료한 쿠버네티스 릴리스 번들
  • 정의 — VMware(Broadcom)가 검증한 K8s 바이너리 + OS 이미지 + 필수 애드온이 하나로 패키징된 릴리스.
    • 포함 애드온 예: CoreDNS · CNI · CSI 등
  • 가치 — 최신 K8s 환경을 안정적으로 배포하고, 클러스터 간 버전·구성 파편화를 방지한다.
  • 운영 — VKS Service가 VKR을 기준으로 게스트 클러스터를 프로비저닝·업그레이드 → 검증된 조합만 배포되어 신뢰성 확보.

9.1 신규 기능 — 아키텍처 & 네트워킹

Zone 구조 · 엣지 노드 제거 · 멀티 네트워크 · 노드 커스터마이징
  • Enhanced Zone Architecture — Zone당 여러 클러스터 배포 → 무중단 하드웨어 교체·수명주기 관리. AI 전용 GPU 등 Custom Zone으로 워크로드 격리.
  • Distributed Transit Gateway (DTGW) — ESXi 호스트가 분산 브리징으로 물리 스위치에 직접 연결. 병목이던 전용 NSX Edge 노드 클러스터 불필요 → Line-rate 성능 확보.
  • Multi Network for VKS Cluster — 테넌트당 다중 Transit Gateway·외부 연결. 분산 VLAN·격리 VPN·정적 라우팅·맞춤 NAT 지원.
  • Node Customization — TuneD(커널·HW 최적화) & AppArmor(컨테이너 보안) 프로파일 적용.
  • CNI Selection & Networking Addon — 프로비저닝 시 특정 CNI 선택 + 추가 네트워킹 애드온 구성.
